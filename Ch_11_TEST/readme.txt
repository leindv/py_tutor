В файле name_function.py написана функция, которая отображает имя и фамилию.
Для проверки правильности написания функции создан файл, в котрый была импортирована 
get_formatted_name() из модуля name_function.
Однако в стандартной библиотеке Python есть модуль unittest, позволяющий теститровать код.
МОДУЛЬНЫЙ ТЕСТ позволяет тестировать правильность работы одного конкретного аспекта поведения функции.
ТЕСТОВЫЙ СЦЕНАРИЙ представляет совокупность модульных тестов, которые совместно доказывают, что 
функция ведет себя так, как положено, во всем диапазоне ситуаций, которые Однакодолжна обрабатывать.
Тестовые сценарии с ПОЛНЫМ ПОКРЫТИЕМ включают полный спектр модульных тестов, покрывающих все возможные
варианты использования функции.
Прохождение тестов представлены в файлах последующих коммитов.
